# BetterBugs トラブルシューティング - リンクがクリックできない問題

## 問題

BetterBugsをインストール後、すべてのリンクがクリックできなくなった。

## 考えられる原因

1. **BetterBugs拡張機能がページのイベントリスナーを妨害している**
2. **拡張機能のバグや設定の問題**
3. **他の拡張機能との競合**
4. **ページの読み込みが完了していない**

## 解決方法

### 方法1: BetterBugsを一時的に無効化して確認

1. **Chromeの拡張機能ページを開く**
   - アドレスバーに `chrome://extensions/` と入力
   - または、メニュー → その他のツール → 拡張機能

2. **BetterBugsを無効化**
   - BetterBugsのトグルスイッチをOFFにする

3. **ページを再読み込み**
   - リンクがクリックできるか確認

4. **結果を確認**
   - リンクがクリックできる場合 → BetterBugsが原因
   - リンクがクリックできない場合 → 他の原因の可能性

### 方法2: BetterBugsの設定を確認

1. **BetterBugsの設定を開く**
   - 拡張機能アイコンを右クリック
   - 「オプション」または「設定」を選択

2. **オーバーレイ機能を無効化**
   - ページ上にオーバーレイを表示する機能がある場合、無効化

3. **ホットキーの設定を確認**
   - リンククリックと競合するホットキーが設定されていないか確認

### 方法3: シークレットモードで確認

1. **シークレットウィンドウを開く**
   - `Ctrl + Shift + N` (Windows) / `Cmd + Shift + N` (Mac)

2. **BetterBugsを有効化**
   - 拡張機能ページで「シークレットモードで許可」を有効化

3. **問題のページにアクセス**
   - リンクがクリックできるか確認

### 方法4: 他の拡張機能との競合を確認

1. **すべての拡張機能を無効化**
   - `chrome://extensions/` で全ての拡張機能をOFF

2. **BetterBugsのみを有効化**
   - BetterBugsだけをONにする

3. **リンクがクリックできるか確認**
   - クリックできる場合 → 他の拡張機能との競合
   - クリックできない場合 → BetterBugsの問題

### 方法5: BetterBugsを再インストール

1. **BetterBugsを削除**
   - `chrome://extensions/` でBetterBugsを削除

2. **ブラウザを再起動**

3. **BetterBugsを再インストール**
   - [Chrome Web Store](https://chromewebstore.google.com/detail/betterbugs-a-fresh-approa/mdljmlgokccncglfobogbfjgcijldnaj)から再インストール

### 方法6: ブラウザのキャッシュをクリア

1. **Chromeの設定を開く**
   - `chrome://settings/`

2. **プライバシーとセキュリティ**
   - 「閲覧履歴データの削除」をクリック

3. **キャッシュされた画像とファイルを削除**
   - 期間: 「全期間」
   - 「キャッシュされた画像とファイル」にチェック
   - 「データを削除」をクリック

4. **ブラウザを再起動**

## 緊急時の対処法

### BetterBugsを無効化して作業を続ける

1. **拡張機能を無効化**
   - `chrome://extensions/` でBetterBugsをOFF

2. **作業を続行**
   - リンクがクリックできるようになる

3. **BetterBugsが必要な時だけ有効化**
   - バグ報告時のみ有効化
   - 報告後、無効化

## BetterBugsの代替機能

BetterBugsが使用できない場合、以下の方法でバグ報告を続けられます：

### 1. ブラウザの標準機能を使用

- **スクリーンショット**: `Windows + Shift + S` (Windows) / `Cmd + Shift + 4` (Mac)
- **画面録画**: Windows標準の画面録画機能
- **コンソールログ**: 開発者ツール（F12）からコピー

### 2. 他のバグ報告ツール

- **Lightshot**: スクリーンショットとアノテーション
- **ShareX**: スクリーンショット、録画、アップロード
- **Snipping Tool**: Windows標準のスクリーンショットツール

## BetterBugsサポートへの問い合わせ

問題が解決しない場合、BetterBugsのサポートに問い合わせ：

1. **BetterBugsのドキュメント**
   - [BetterBugsドキュメント](https://docs.betterbugs.io/)

2. **GitHub Issues**
   - BetterBugsのGitHubリポジトリがあれば、Issueを作成

3. **サポートメール**
   - BetterBugsの公式サイトから問い合わせ

## 現在の403エラー調査への影響

BetterBugsが使用できない場合でも、調査は続けられます：

1. **スクリーンショット**: ブラウザの標準機能を使用
2. **ログの確認**: サーバー側で直接確認
3. **問題の報告**: スクリーンショットとログを手動で共有

## 推奨される対処順序

1. **方法1**: BetterBugsを一時的に無効化して確認（最も簡単）
2. **方法2**: BetterBugsの設定を確認
3. **方法5**: BetterBugsを再インストール
4. **方法6**: ブラウザのキャッシュをクリア
5. **方法4**: 他の拡張機能との競合を確認

## 次のステップ

1. **方法1を試す**: BetterBugsを無効化して、リンクがクリックできるか確認
2. **問題が解決した場合**: BetterBugsの設定を調整
3. **問題が解決しない場合**: BetterBugsを再インストール、またはサポートに問い合わせ
